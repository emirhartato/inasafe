Mendownload OpenStreetMap

InaSAFE mendukung penggunaan data bangunan dalam berbagai analisis fungsi dampak, khususnya data bangunan yang bersumber dari OpenStreetMap <https://openstreetmap.org> _ proyek (OSM). Alat ini membutuhkan koneksi internet untuk mengambil data melalui layanan berbasis web yang berjalan pada Linfiniti’s OSM Reporter situs web.

Data, setelah didownload akan tersedia untuk Anda sebagai sebuah shapefile. Sebuah gaya file secara otomatis dibuat sehingga dilambangkan dengan baik dalam QGIS. Selain itu, metadata kata kunci yang benar dibuat untuk dataset didownload sehingga dapat digunakan secara langsung dalam skenario analisis dampak menggunakan InaSAFE.

Catatan: Alat mendownload data bangunan memiliki batas untuk ukuran dataset yang akan diambil. Mungkin Anda memiliki masalah ketika ingin mengambil wilayah sebuah negara. Umumnya dataset untuk tingkat kota dan dibawahnya akan berjalan dengan baik.

Untuk menggunakan alat ini secara efektif:

  • Gunakan QGIS untuk memperbesar ke daerah yang data bangunannya akan diambil.

  • Periksa direktori output yang benar. Perhatikan bahwa dataset tersimpan akan bernama buildings.shp (begitu pun file terkait lainnya).

  • Tekan OK Button dan tunggu data untuk diambil.

Peringatan

Jika dataset sudah ada di direktori output, dataset tersebut akan ditimpa. Alat ini memerlukan koneksi internet untuk bekerja dan mengambil bangunan akan menggunakan bandwidth Anda.

Catatan

Data yang didownload adalah hak cipta kontributor OpenStreetMap (klik untuk info lebih lanjut).
