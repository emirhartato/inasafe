Pilihan¶

InaSAFE plugin menyediakan dialog pilihan yang memungkinkan Anda untuk menentukan berbagai pilihan yang berhubungan untuk bagaimana InaSAFE akan berperilaku. Dialog pilihan dapat ditampilkan dengan mengklik pada InaSAFE simbol plugin toolbar pilihan (seperti yang terlihat di bawah) atau dengan melakukan Plugins ‣ InaSAFE ‣ InaSAFE Options.

Catatan

Anda dapat klik tombol Help beberapa waktu dan ini akan membuka bantuan pencarian dokumentasi untuk halaman ini

Berikut ini pilihan yang tersedia di dalam Options Dialog:

  • Only show visible layers in the InaSAFE dock : Pilihan ini akan menentukan apakah semua (ketika tidak dicentang) layer ancaman dan dampak harus terdaftar dalam kotak kombo dock. atau (ketika dicentang) hanya layer yang terlihat.

  • Set QGIS layer name from ‘title’ in keywords : Pilihan ini akan (ketika diaktifkan) mengakibatkan untuk layer nama pada Layers tree menggunakan kata kunci title di file kata kunci layer. Jika layer tidak memiliki ‘title’ pada kata kunci, atau tidak memiliki kata kunci sama sekali, perilaku QGIS normal untuk penamaan layer yang akan diterapkan.

  • Zoom to impact layer on scenario estimate completion : Pilihan ini akan menyebabkan tampilan peta untuk diperbesar/diperkecil sehingga benar-benar berisi InaSAFE hasil skenario peta dampak ketika analisis selesai.

  • Hide exposure layer on scenario estimate completion : Pilihan ini akan menyebabkan QGIS untuk mematikan layer keterpaparan yang sedang digunakan ketika InaSAFE melengkapi analisis ini. Anda dapat mengaktifkan kembali visibilitas layer lagi dengan mencentang kotak centang di legenda.

  • Clip datasets to visible extent before analysis : Pilihan ini akan menyebabkan QGIS memotong layer ancaman dan keterpaparan yang terlihat pada peta kanvas.

  • When clipping, also clip features (e.g. will clip polygon smaller): Pilihan ini, setiap poligon yang melampaui luasan analisis akan dipotong sehingga mereka berisi oleh luasan analisis. contohnya jika Anda memiliki layer poligon ancaman banjir yang meluas jauh melampaui area yang Anda minati, poligon banjir akan berkurang sehingga hanya bagian di dalam area yang Anda minati dipertahankan. Ini dapat mempercepat waktu pemrosesan.

  • Help to improve InaSAFE by submitting errors to a remote server : Pilihan ini, ketika diaktifkan, akan mengirimkan informasi diagnosis kembali ke server proyek InaSAFE di setiap kesalahan kita mampu menjebaknya. Informasi yang diberikan berguna bagi tim InaSAFE untuk meningkatkan ketahanan perangkat lunak yang kami produksi dan untuk melihat tren pada isu-isu yang dihadapi orang. Silakan catat teks peringatan dibawah pilihan ini yang disertakan disini secara penuh:

Catatan

Pengaturan di atas memerlukan Anda untuk me-restart QGIS setelah menonaktifkan/mengaktifkan perintah tersebut. Pesan kesalahan dan informasi diagnosis akan dipublikasi ke http://sentry.linfiniti.com/inasafe-desktop/. Beberapa institusi tidak memungkinkan Anda untuk mengaktifkan fitur ini - periksa dengan jaringan administrator jika tidak yakin. Walaupun data dikirim secara anonim, informasi yang terisi di traceback mungkin berisi path sistem file yang mengungkapkan identitas Anda atau informasi lain dari sistem Anda.

  • Show intermediate layers generated by postprocessing : Pilihan ini akan mnyebabkan QGIS menampilkan file menengah yang dihasilkan oleh langkah-langkah postprocessing pada peta kanvas.

  • Default female ratio : Ini adalah standar wantia untuk jumlah populasi rasio yang menghancurkan digunakan por menghasilkan jenis kelamin

  • Keyword cache for remote datasources : This option is used to determine where keywords are stored for datasets where it is not possible to write them into a .keywords file. See Sistem Kata kunci in the InaSAFE User-Manual for more information on the keywords system.
  • Logo for maps (must be x x y): This option is used to determine a custom logo that will be used in the pdf of your impact scenario project. See Reports in the InaSAFE User-Manual for more information on the reports. NOTE: image should have size X x Y.
  • Report template: This option is used to determine a directory with custom templates. Templates from this directory will be available for usage when generating report on your impact scenario project. See Reports in the InaSAFE User-Manual for more information on the reports.

  • Enable developer mode. Dengan diaktifkannya pilihan ini, Anda dapat meng-klik kanan pada laporan dan menggunakan alat debug webkit untuk melakukan debug pada hasil. Opsi ini sebaiknya digunakan untuk pengguna tingkat lanjut saja.

Catatan

Pressing Cancel at any time will close the options dialog and any changes made will not be applied. Pressing OK at any time will close the options dialog and any changes made will be applied immediately.

Catatan

Urutan tombol yang tepat yang ditampilkan pada kotak dialog ini mungkin berbeda tergantung pada sistem operasi atau lingkungan desktop Anda.
